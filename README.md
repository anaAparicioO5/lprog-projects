## Descripción General

Este repositorio agrupa diferentes proyectos y ejercicios, cada uno enfocado en conceptos clave de programación:

- Paradigmas: funcional, lógico, orientado a objetos, concurrente.
- Estructuras de datos: listas, árboles, grafos, colas, pilas, tablas hash.
- Patrones y buenas prácticas: separación de responsabilidades, modularidad, pruebas automatizadas.
- Herramientas de desarrollo: linters, formateadores, gestores de dependencias.


## Tecnologías y Lenguajes

- Prolog (lógica)
